Qore Programming Language

Embeddable Multithreaded Programming Language

Qore is an interpreted scripting language with:

  • multithreading and thread-safe programming
  • strong typing and exception handling
  • code embedding and sandboxing
  • automatic memory management and c++-style destructors

Why Use Qore?

  • High-level APIs for SQL generation, schema management, and DBA activities
  • Extensive support for client/server HTTP services with a focus on data-centric REST
  • Fundamental support for interfacing, data serialization, data transformations, file- and message-based data transfers
  • Fundamental support for Test-Driven Development
  • permissive open source license (choice of MIT, LGPL, or GPL)
  • runs on UNIX/Linux, macOS, and Windows
